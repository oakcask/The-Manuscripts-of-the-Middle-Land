# 曖昧性の分類軸

神話語彙は、単なる多義語ではなく、失われた世界認識を人類語へ押し込んだ結果として曖昧になる。

## 解釈層

| 層 | 説明 |
|---|---|
| 神話層 | 神、船、空、土、光などとして語られる宗教的・叙事的な意味。 |
| 物理層 | 宇宙論、重力、観測限界、エントロピー、情報損失として読める意味。 |
| 技術層 | アップロード、同期、航行、保存、複製、検証として読める意味。 |
| 政治層 | 検閲、共同体維持、異端処理、正統化として読める意味。 |
| 翻訳層 | 翻訳者AIが人類語に置換したことで発生した意味の歪み。 |

## 曖昧性の種類

| 種類 | 説明 |
|---|---|
| 指示対象の揺れ | 語が場所、装置、人物、制度、現象のどれを指すか判定できない。 |
| 価値の反転 | 救済を意味する語が、別写本では破壊や処刑を意味する。 |
| 時制のずれ | 過去、未来、未観測、到達不能、喪失済みが同じ語で語られる。 |
| 身体性の欠落 | 人、母、血、土などが生物的意味を失っている可能性がある。 |
| 検閲の痕跡 | 危険な語が柔らかい宗教語彙や自然語彙に置換されている。 |
| 翻訳者の汚染 | 翻訳者AIが人類の神話語彙で原義を上書きしている。 |

## 注釈で確認する問い

- この語は何を隠しているか。
- この語は誰にとって都合がよいか。
- この語を別の写本ではどう呼んでいるか。
- この語の意味が反転すると、神話本文はどう読めるか。
- 翻訳者AIはこの語を説明しすぎていないか、または避けすぎていないか。
