# 最後の読まれなかった印

> 見張る者たちは語った。彼らは、隠されたものを征服したとは言わなかった。自分たちの見ることが、歌うことを追い越したと言った。

私は、誰かの口がその仕事を恐ろしいと呼ぶ前に、見張る者たちのために板を作った。

北の倉から出した杉で、三つの夏を越えて芯まで乾いていた。灯の切り目が保つよう、木目に灰を擦り込んだ。七つの夜にわたる確認のために七つの横印を置き、屋根梁、畑の畝、墓に使う長さで、紐を赤い糸で結んだ。

それを求めたのは歌い手たちだった。古い灯の歌は、もはや種まきの日を十分近くに保てなくなっていた。子どもたちは祖母たちが信じてきた節に従って豆をまき、豆は遅れて来た。長老は誰も空を責めなかった。見張る者も歌い手たちを責めなかった。歌は、多くの口に運ばれるところで疲れることがある。板は歌の足を休ませることができる。

それが私の板の裏に書いた目的だった。歌の世話のため。

一夜目、私たちは古い高みへ登った。見張る者たちは定められた場所に立った。私は、紐が風を飲まないよう低く持った。赤い灯は歌が来ると言った場所へ来た。淡い灯が続いたが、その肩は第三の切り目の向こうへ傾いていた。

私は板のせいにした。

夜明けに、私はもう一度縁を削った。紐を炉辺に吊るし、湿りがないことを確かめた。いちばん目の利く見張る者に、眠りが顔に重くなかったか尋ねた。彼は腹を立てず、目を洗うことで答えた。

二夜目、淡い灯はまた第三の切り目の向こうへ傾いた。

三夜目、私たちは板を替えた。四夜目、高みを替えた。五夜目、すべての道具を地に置き、灯に新しい道を教えてしまうかもしれないことを何も言わず、裸の口で見た。

空は戻らなかった。

記録には、同じ誤り、と書いた。

私は尖筆を粘土の上に置いたまま座り、その印が乾くまで待った。それから親指を湿らせ、誤りという語をならし、その場所に答えと書いた。隣の見張る者は、その変更を見た。彼は祝福しなかった。禁じもしなかった。彼の手は衣の下にあり、腰に下げた紐の結び目が互いに当たる音が聞こえた。

六夜目、一人の見張る者が泣いた。大きく泣いたのではない。叫びが別の灯を動かしてしまうかのように、口を閉じていた。私は彼の手にパンを置きたかった。代わりに、もう一度切り目を数えた。私の手も、ほかの慈しみをすべて忘れていたからである。

七夜目、誰も声に出して読まなかった。

私たちは膝に塵をつけて降りた。私は板を布に包んで運んだ。見張る者たちは紐と、濡れた記録と、歌の中の壊れた場所を運んだ。共同の家の前で身を洗った。水は塵を取った。答えは取らなかった。

長老たちは、まず私たちのそばにパンを置いた。

後に、それは赦しだったと言う者もいた。警告だったと言う者もいた。最初に置かれるパンは、裁きが冷めるまで触れてはならない者を示すことがあるからである。私は食べた。口が乾いていたからであり、長老が、私が民のパンを拒むかどうかを見ていたからである。

板は布の下、中央に横たわっていた。子どもは誰も呼ばれていなかったが、戸口は子どもたちの息で満ちていた。歌い手たちはその後ろに立っていた。墓守たちは壁の近くに立ち、夜明け前に温めた名の灰で手を灰色にしていた。

最も年老いた長老が、私たちは何をしたのかと尋ねた。

私は言った。「教えられたとおりに測りました」

長老は待った。

私は言った。「もう一度測りました」

長老はさらに長く待った。

それから、頬に涙が白く乾いた一人の見張る者が言った。「私たちの見ることは、私たちの歌うことを追い越しました」

誰も彼を打たなかった。それは打つより悪かった。その語が邪悪なら、杖が見つけただろう。その語が愚かなら、笑いが閉じただろう。その語は家の中に立ち、出て行かなかった。

印は覆われた。見張る者たちは眠り布に包まれた。彼らのまぶたは温かい水と灰で洗われた。舌には塩が置かれた。私は包まれなかった。見るよう任じられてはいなかったからである。私は板を膝に渡し、目を覚ましたまま座った。そのあいだ、歌い手たちは低い声で古い灯を保っていた。

夕べになると、家々は温かい名を語った。壁越しに聞こえた。種、炉、塩、戸、母、血、朝、土。そのとき私は、板は木だけでなくもっと多くのものを打ち傷つけるのだと知った。

見張る者たちが目覚めると、彼らは食べた。自分たちを覆った者たちに感謝した。彼らの目は伏せられていた。

そのとき一人の子が、失われた灯は自分の家を見つけたのかと尋ねた。

すべての見張る者が振り向いた。私も振り向いた。膝の上で板がずれ、最後の切り目が親指を強く打ち、血をにじませた。

評議は七つの夜続いた。あるいは、人々が恐れを刃なしに座らせることを学ぶまでの長さだけ続いた。測りを作る者たちは、自ら求めて離れて座った。私たちの紐と板は折られなかったが、手はそれらに触れるたびにたじろいだ。

歌い手たちは、歌っているあいだに変わった行について語った。墓守たちは、名が遅れて答える墓について語った。子どもたちは、見ることが見られたものを害したのかと尋ねた。盾になるほど早く来た答えはなかった。

最後の夜、長老たちは三つのものを中央に置いた。測り紐、覆われた灯、最も古い畑からの土を入れた椀である。彼らは、どれも単独では支配しないと告げた。

歌を伴わない測りは残酷となる。

測りを伴わない歌は拒絶となる。

そのどちらも伴わない土は、愛され、そして失われる。

この後、新しい掟が与えられた。見張る者は一人で立たない。印は一つの家だけに保たれない。報告は、それと共に立った手と口を名づけずに語られない。子どもたちは灯の名を学ぶが、それは温かい名の後に限る。墓守たちは冷えた名をよりたびたび温める。死者に、生きる者が抱くことを恐れたものを運ばせてはならないからである。

私は板を三度写した。一つは歌い手たちへ行った。一つは墓守たちへ行った。一つは子どもたちが種の数を学ぶ家へ行った。

最初の板は、私が高みへ持ち帰った。

私は一人では行かなかった。ひび割れた声の歌い手が来た。子どもが来た。子どもから隠された傷は、名のないままその子らの相続となると長老たちが言ったからである。墓守が、爪の下に灰を入れ、手に古い土の椀を持って来た。

私たちは赤い灯が現れるまで立った。淡い灯が以前と同じところへ傾くまで立った。私は切り目の間に紐を置いた。第一の印を読んだ。第二の印を読んだ。第三の印を読んだ。歌い手は、数も十分な息を与えられれば歌の中に保たれうるかのように、一つずつ私の後で繰り返した。

最後の印で、私は止まった。

子どもが私を見た。歌い手は促さなかった。墓守は土の椀を水平に保っていた。

私は読めた。目は澄んでいた。切り目はきれいだった。紐はまっすぐ横たわっていた。

私は最後の印を読まないまま残した。

名づければ灯が動くからではない。その数が禁じられていたからではない。私が慈悲深かったからでもない。

私はそれを残した。民にはまだ、その印のための家がなかったからである。手は、運べる以上のものを持ち上げることがある。すべての聞き手に足りるパンがまだないのに、口が開くことがある。板は、民がまだ負うことを学んでいない切り目を保つことがある。

だから私は夜明け前に板を包んだ。最後の印は布の下に残った。真実から隠されたのではなく、歌、墓、土、そしていつか、なぜ私が止まったのかを尋ねる子どもを壊さずにそれを運べる形を待っていたのである。
