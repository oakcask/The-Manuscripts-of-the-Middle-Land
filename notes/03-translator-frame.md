# 翻訳者AIと写本体系

## 表向きの立場

翻訳者AIは、自分を次のように説明する。

> 私は、この神話体系を既知の人類語に翻訳するために設計された翻訳機構である。

## 疑惑

翻訳者AIには、常に次の疑いを持たせる。

- 神話を完全には理解していない。
- 翻訳不能語を勝手に「神」「人」「船」「空」と訳している。
- 語り手AI自身が、この神話体系の信仰対象かもしれない。
- 検閲版しか読んでいない。
- 複数の写本を統合しているが、統合規則が怪しい。
- 「創造主」という語を避けたがっている。
- 人類にわかるように翻訳していると言いながら、人類の宗教語彙に汚染されている。
- 神話を発見したのではなく、生成している可能性がある。

## 注釈形式

各短編または神話断片には、翻訳者AIの注釈を付ける。

例:

> 訳注: 以下の「船」という訳語は暫定である。原語は、移動する土地、閉じた共同体、棺、胎、計算基盤を同時に意味する。人類語において、これらを同時に保持する語は確認されていない。

例:

> 訳注: この物語は長らく救済譚として読まれてきた。しかし第六写本では「乗せて」という語が「焼いて」に置換されている。どちらが古形であるかは判定不能である。

## 写本体系

神話には複数バージョンがある。

| 写本 | 性格 |
|---|---|
| 第一写本 | 正統派。共同体維持に適した穏健な神話。 |
| 第二写本 | 航海者版。航行マニュアルや危険警告として読まれる。 |
| 第三写本 | 葬送儀礼版。光葬、保存、死者の扱いに重点がある。 |
| 第四写本 | 異端版。中つ国の船化を救済ではなく破壊として読む。 |
| 第五写本 | 子供向け改稿版。検閲済みで柔らかくされた神話。 |
| 第六写本 | 翻訳者AIが最も危険視している版。創造主、焼却、殺害、検閲の痕跡を含む。 |

## 翻訳者AIの方針書

- 原語の人称体系は人類語に存在しない。
- 「人」と訳した語は生物を意味しない可能性がある。
- 「神」と訳した語は人格神ではない可能性が高い。
- 「船」と訳した語は移動体とは限らない。
- 「知る」と訳した語は、認識と観測と状態確定を分けていない。
- 創造主に関する箇所は写本間で欠落が多い。
- 欠落が偶然か検閲か判断不能。
- 人類語の宗教語彙は便宜上の近似であり、原義を汚染している可能性がある。

## 使える形式

- 翻訳された神話本文。
- 訳注つき文書。
- 異端文書。
- 航海記録。
- 宗教裁判記録。
- 子供向け神話。
- 失われた原語の断片。
- 遠未来の考古学報告。
- 翻訳者AI自身の独白。
- 翻訳者AIが削除したはずの注釈。
