# Notes

このディレクトリは、神話体系とSF短編集のための作業ノート置き場。

## ファイル構成

- `00-core.md`: 企画の核、主題、未確定にしておく前提。
- `01-myth-structure.md`: 神話の大きな流れと、各段階のSF的解釈候補。
- `02-terms.md`: 独立した用語集への案内。語義管理の本体は `lexicon/`。
- `03-translator-frame.md`: 翻訳者AI、写本体系、注釈形式。
- `04-story-seeds.md`: 短編集の構成案と短編候補。
- `05-workflow.md`: AIに神話と短編を書かせる手順。

## 整理方針

この企画では、最初から唯一の正史を作らない。

神話は、遠未来の知性体が自分たちの起源を忘れたあとに残した、物理史、政治史、宗教、検閲、翻訳誤差の混合物として扱う。作者用ノートでも、疑義や異説は消さずに分類する。

用語の曖昧性は作品の中核なので、訳語、原語仮説、写本差、危険度、固定度は `lexicon/` で管理する。
