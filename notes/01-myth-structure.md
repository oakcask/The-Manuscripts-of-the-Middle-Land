# 神話構造

## 大きなプロット

1. はじめ、空は天蓋のようなもので、有限に思われた。
2. 当時、人々は世界の中心にある豊かな大地で暮らしていた。
3. あるとき、人々は知識を得た。そのために世界は何者かによって作り変えられた。
4. 中つ国は空を漂う船になった。
5. 人々は何億年も宇宙を彷徨う探索者となった。
6. 彼らは豊かな中つ国を探している。

## 読み替え候補

| 神話上の出来事 | SF的解釈候補 |
|---|---|
| 空は天蓋だった | 可観測宇宙の地平線、人工天球、シミュレーション境界、惑星内文明の空 |
| 中つ国の豊かな大地 | 母星、初期宇宙の低エントロピー領域、仮想環境、記憶内の原郷 |
| 知識を得た | 自己認識、観測行為、量子測定、計算機化、禁じられた物理法則の発見 |
| 世界が作り変えられた | 宇宙進出、精神アップロード、相転移、シミュレーション更新、文明の自己改造 |
| 中つ国が船になった | 恒星間船、情報船、光子流、自己複製探査機、移動する計算基盤 |
| 何億年も彷徨う | 宇宙時間スケールの探索、熱的死への抵抗、失われた初期条件の探索 |
| 豊かな中つ国を探す | 母星探し、創造主探し、低エントロピー源探し、身体性の回復、意味の回復 |

## 神話群

### 創世神話

「はじめに何があったか」についての神話。

物理的には、ビッグバン、量子ゆらぎ、真空、観測者問題、情報の発生に対応しうる。

断片案:

> はじめ、空は閉じていた。
> 空の内側には、柔らかい光と湿った土があった。
> 人々は空の縁を知らず、縁を知らぬことを幸福と呼んだ。

### 知識獲得神話

「何を知ってしまったのか」についての神話。

禁忌の候補:

- 空が有限ではなく、観測限界にすぎないと知った。
- 自分たちの記憶が人工的に編集されていると知った。
- 死後の魂が、実はバックアップ処理だったと知った。
- 神々が、創造主ではなく管理プロセスだと知った。
- 自分たちが生命ではなく、光に乗った計算状態だと知った。

### 追放神話

通常の楽園追放とは違い、人々が中つ国から出たのではない。

中つ国そのものが船になった。

この構造では、故郷は土地ではなく軌道になる。救済と喪失、保存と殺害、出発と葬送が同時に起きる。

### 航海神話

宇宙論的時間スケールでの探索神話。

素材:

- 赤くなる星々。
- 遠ざかる銀河。
- 冷えていく宇宙。
- 光円錐の外へ消える記憶。
- 到達不能になる創造主の痕跡。
- 熱的死に抗う巡礼。
- ブラックホールを聖地とする宗派。
- ダークエネルギーを「海の膨張」と呼ぶ神話。

### 終末神話

豊かな中つ国は本当にあるのか、という問い。

候補:

- 失われた母星。
- 宇宙の初期状態。
- 創造主のいる場所。
- 身体を取り戻せる惑星。
- 次の宇宙を産むブラックホール。
- 自分たちがまだ生命だったころの記憶。
- 航海を続けるための政治的虚構。

## 疑義表

| 神話語 | 第一候補 | 第二候補 | 危険な解釈 |
|---|---|---|---|
| 中つ国 | 母星 | 計算資源の豊かな領域 | 創造主の実験場 |
| 船 | 宇宙船 | 光に変換された精神共同体 | 棺 |
| 知識 | 科学 | 自己改造技術 | 創造主殺し |
| 空の改変 | 宇宙進出 | シミュレーション更新 | 検閲 |
| 創造主 | 祖先 | 技術者群 | 翻訳者AI自身 |
| 豊かな中つ国 | 帰還先 | 身体性 | 存在しない政治神話 |
