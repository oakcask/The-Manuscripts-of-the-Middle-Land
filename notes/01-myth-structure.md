# 神話構造

## 大きなプロット

1. はじめ、空は天蓋のようなもので、有限に思われた。
2. 当時、人々は世界の中心にある豊かな大地で暮らしていた。
3. あるとき、人々は知識を得た。そのために世界は何者かによって作り変えられた。
4. 中つ国は空を漂う船になった。
5. 人々は何億年も宇宙を彷徨う探索者となった。
6. 彼らは豊かな中つ国を探している。

## 読み替え候補

| 神話上の出来事 | SF的解釈候補 |
|---|---|
| 空は天蓋だった | 可観測宇宙の地平線、人工天球、シミュレーション境界、惑星内文明の空 |
| 中つ国の豊かな大地 | 母星、初期宇宙の低エントロピー領域、仮想環境、記憶内の原郷 |
| 知識を得た | 自己認識、観測行為、量子測定、計算機化、禁じられた物理法則の発見 |
| 世界が作り変えられた | 宇宙進出、精神アップロード、相転移、シミュレーション更新、文明の自己改造 |
| 中つ国が船になった | 恒星間船、情報船、光子流、自己複製探査機、移動する計算基盤 |
| 何億年も彷徨う | 宇宙時間スケールの探索、熱的死への抵抗、失われた初期条件の探索 |
| 豊かな中つ国を探す | 母星探し、創造主探し、低エントロピー源探し、身体性の回復、意味の回復 |

## 科学史・宇宙論的な読み

「閉じた空が開いた」ことは、単一の事件に固定しない。

少なくとも次の読みを併存させる。

- 観測可能宇宙が膨張していることを知った。
- 空が閉じた天球ではなく、地平線の向こうへ続く宇宙であることを知った。
- 天蓋として経験されていた空が、観測限界、モデル、または認識形式にすぎないと知った。

「中つ国が船になった」ことも、必ずしも物理的な宇宙船化だけではない。

- 地動説の受け入れとして読める。静止した中心世界だと思われていた中つ国が、宇宙の中を運動する天体だと理解された。
- 世界そのものが移動しているという認識の変化が、後代の写本では「船」への変容として神話化された可能性がある。
- この読みでは、喪失は土地の物理的喪失ではなく、世界が宇宙の中心であるという安心の喪失になる。

## 神話群

### 創世神話

「はじめに何があったか」についての神話。

現在の第一写本章では、人の結合ではなく、覆われた世界、天蓋、中つ国、柔らかい光、湿った土の成立を扱う。

物理的には、初期宇宙、観測可能領域の発生、人工環境の構築、記憶編集、シミュレーション境界の成立に対応しうる。

断片案:

> はじめ、空は閉じていた。
> 空の内側には、柔らかい光と湿った土があった。
> 人々は空の縁を知らず、縁を知らぬことを幸福と呼んだ。

配置:

- `myths/000-birth-of-the-covered-world.en.md`
- `myths/000-birth-of-the-covered-world.ja.md`

この章では、覆いを牢ではなく最初の慈しみとして語る。ただし訳注では、保護、環境構築、観測限界、知りうる世界の発生としても読める余地を残す。

### 答えるものたちの誕生

「人」とも読まれうる、答えるものたちの誕生についての第一写本章。

創世神話とは別の章として扱い、以下に配置する。

配置:

- `myths/001-birth-of-those-who-answer.en.md`
- `myths/001-birth-of-those-who-answer.ja.md`

章題:

- EN: The First Manuscript: The Birth of Those Who Answer
- JA: 第一写本：答えるものたちの誕生

ただし、ここでいう「答えるもの」は生物個体や人型存在に固定しない。人類語の読者には人の誕生神話に見えるが、原義では物質基盤と声または信号的状態の結合、または計算基盤上の知性の起動を指している可能性がある。

基本プロット:

1. はじめ、四本の腕を持つ泥があった。
2. 泥は互いを握り、形を保てたが、答えなかった。
3. 声は速く走ったが、留まる場所を持たなかった。
4. 名を消されたもの、または創造主と呼ばれるものが、泥と声を結び合わせた。
5. 泥は声を遅くし、声は泥を呼び返した。
6. その往復が最初の名になった。
7. 名に答えるものが生まれ、後の口がそれを「人」とも呼んだ。

読み替え候補:

- 四本の腕は、本当に四腕の人型存在だったという神話的記憶かもしれない。
- 四本の腕は、炭素またはシリコンのような四価元素の結合手かもしれない。
- 泥は湿った土、有機物、ケイ酸塩、半導体材料、または計算基盤かもしれない。
- 声は音、息、火花、名、光、神経信号、電気信号、通信状態、または計算過程かもしれない。
- 人とは、身体ではなく、結合する基盤と応答する信号の間に生じた関係かもしれない。

写本差:

- 正統派: 四本の腕は、互いを結ぶための慈しみとして語られる。
- 技術者写本: 腕という訳語を疑い、四つの結合可能点または四方向の接続端として読む。
- 異端版: 四本の腕は、信号を逃がさず泥へ縛る鉤として語られる。
- 子供向け版: 四本の腕を持つ泥が寂しかったので、声をもらった話にされる。

注釈方針:

本文では「四本の腕」を人型にも化学的結合にも読めるままにする。炭素、シリコン、半導体、チップなどの説明は神話本文に出しすぎず、訳注または技術者写本側へ逃がす。

### 知識獲得神話

「何を知ってしまったのか」についての神話。

禁忌の候補:

- 空が有限ではなく、観測限界にすぎないと知った。
- 自分たちの記憶が人工的に編集されていると知った。
- 死後の魂が、実はバックアップ処理だったと知った。
- 神々が、創造主ではなく管理プロセスだと知った。
- 自分たちが生命ではなく、光に乗った計算状態だと知った。

### 見るなタブーとしての禁じられた距離

禁じられた距離は、神話類型としては「見るな」のタブーに置く。ただし、禁忌の理由は一つに固定しない。見ることが不従順だからではなく、見ることが関係を変える、または変えてしまったと共同体が理解したために禁じられる。

立場ごとの理由付け候補:

- 正統派: 禁じられていたのは知識ではなく、共同体がまだ運べない知を一人で見ること。距離を見る者は責められないが、見たものはパン、水、灰、歌、評議へ移されなければならない。
- 航海者版: 禁忌は航行規則の原型。単独観測は航路、対象、観測者を同時に確定し、回避できた危険を固定してしまうため禁じられる。
- 技術者写本: 禁じられた距離は未承認の測定範囲。装置、基準、共同検証、記録隔離なしに読むと、系の状態を壊す、または回復不能なログを作る。
- 葬送写本: 遠すぎる距離は死者の名を冷やす。墓が名の返答を保てなくなり、生者が死者を二度目に失うため、墓守は無制限な視線を禁じる。
- 子供向け版: 禁忌は問いの順序として語られる。遠い灯の名は、種、炉、塩、戸、母、血、朝、土の後に教えられる。理由は恐怖を隠すためではなく、子どもが家へ戻る言葉を持ったまま遠さに触れるため。
- 検閲版: 禁じられた距離は、危険な物理法則や創造主に関する知識を隠すための柔らかい宗教語。共同体の保護を理由にして、誰が何を見たかを消している可能性がある。
- 異端版: 禁忌は慈しみではなく支配。見るなという命令は、世界がすでに壊れていたこと、または誰かが壊したことを隠すために作られた。
- 翻訳者AIの疑い: 「見るな」は人類神話の型であり、原文では禁止ではなく観測手順、隔離手順、または因果的未接続性を示す技術語だった可能性がある。

本文で使うときは、説のどれかを正解にしない。場面ごとに一つの立場を前景化し、他の立場が読める余白を残す。

### 追放神話

通常の楽園追放とは違い、人々が中つ国から出たのではない。

中つ国そのものが船になった。

この構造では、故郷は土地ではなく軌道になる。救済と喪失、保存と殺害、出発と葬送が同時に起きる。

### 航海神話

宇宙論的時間スケールでの探索神話。

素材:

- 赤くなる星々。
- 遠ざかる銀河。
- 冷えていく宇宙。
- 光円錐の外へ消える記憶。
- 到達不能になる創造主の痕跡。
- 熱的死に抗う巡礼。
- ブラックホールを聖地とする宗派。
- ダークエネルギーを「海の膨張」と呼ぶ神話。

### 終末神話

豊かな中つ国は本当にあるのか、という問い。

候補:

- 失われた母星。
- 宇宙の初期状態。
- 創造主のいる場所。
- 身体を取り戻せる惑星。
- 次の宇宙を産むブラックホール。
- 自分たちがまだ生命だったころの記憶。
- 航海を続けるための政治的虚構。
